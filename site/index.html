    Overview¶

    Vous trouverez sur mon site officiel le cours Big Data associé à ces TPs. L'objectif de ce cours est d'initier les étudiants aux architectures Big Data: Les notions, technologies, patrons et bonnes pratiques. Le cours comportera les chapitres suivants:

    • Introduction au Big Data
    • Hadoop et Map Reduce
    • Traitement de données (Batch, Streaming, Temps Réel, etc.)
    • Bases de données NoSQL
    • Putting it all together

    Ce cours comporte cinq séances de travaux pratiques:

    • TP1: Hadoop et Map Reduce
    • TP2: Apache Spark
    • TP3: Apache Kafka
    • TP4: HBase
